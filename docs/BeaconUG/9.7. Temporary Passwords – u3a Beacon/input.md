**9.7.** **Temporary** **Passwords**<img src="./jlf31kuk.png"
style="width:2.05208in;height:0.38542in" />

> Back

All System users are initially assigned unique temporary passwords. A
unique temporary password is also assigned if your password has to be
re-set for any reason. Temporary passwords must be changed at first
login, see [4. Logging
in](https://u3abeacon.zendesk.com/hc/en-gb/articles/360020033698) [with
a new
password](https://u3abeacon.zendesk.com/hc/en-gb/articles/360020033698).
In this way no-one should know your password except yourself. Keep it
somewhere safe but well disguised so that it cannot be accessed by
anyone else.
