**Letters** in Beacon are <u>one page</u> documents that can be
personalised with members’ details (Name, Address, Membership number,
etc.).

Letters may be created from several of the menu selections under the
Membership heading, e.g. the Members List.

*Note:* *the* *letter* *operations* *that* *you* *can* *perform* *may*
*be* *different* *to* *those* *described* *below,* *according* *to*
*the* *access* *that* *has* *been* *allocated* *to* *your* *Role* *by*
*your* *U3A.*

Creating Letters

From any list of members, select the ones that you wish to create a
Letter for and proceed as described in
[**<u>Section</u>**](https://u3abeacon.zendesk.com/hc/en-gb/articles/360007318857)
[**<u>6.2.1</u>**](https://u3abeacon.zendesk.com/hc/en-gb/articles/360007318857).

Standard Letter Templates

If you create letters regularly they can be saved as standard templates
as described in [**<u>Section
6.2.2</u>**](https://u3abeacon.zendesk.com/hc/en-gb/articles/360007380578-6-2-2-Standard-Letters).
