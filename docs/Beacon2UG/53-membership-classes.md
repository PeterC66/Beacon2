# 53. Membership Classes

Membership Classes define the different types of membership your u3a offers —
for example Individual, Joint, Associate, or Honorary. Each class can have its
own fee, and you can control which classes appear on the online joining form.

To open this page, click **Membership classes** in the **Set Up** section of the
Home page.

![The Membership Classes page](images/53-membership-classes.png)

---

## The class list [A]

The table shows each membership class with:

| Column | What it shows |
|--------|---------------|
| **Name** | The name of the class |
| **Current** | Whether this class is for current (active) members |
| **Joint** | Whether this is a joint membership class |
| **Associate** | Whether this is an associate membership class |
| **Fee** | The annual fee for this class |

Classes that are **locked** (because members are assigned to them) cannot be
deleted, though you can still edit their settings.

---

## Editing a class [B]

Click a class name to open the edit form. You can change:

| Field | Purpose |
|-------|---------|
| **Name** | The name shown throughout Beacon2 and on the online joining form |
| **Explanation** | A short description shown to prospective members on the online joining page to help them choose the right class |
| **Current** | Tick if this class represents active membership |
| **Joint** | Tick if this is a joint (partner) membership class |
| **Associate** | Tick if this is an associate membership class |
| **Show online** | Tick if this class should appear as an option on the online joining form |

---

## Setting fees [C]

There are two fee modes:

### Fixed fee

Enter a single **fee per person** that applies all year round. You can also
enter an optional **Gift Aid eligible amount** — this is the portion of the fee
that qualifies for Gift Aid (it may be less than the full fee if part of the
subscription covers services that are not eligible).

### Monthly varying fees

If your u3a charges a different fee depending on which month a member joins,
switch to **monthly varying fees**. This shows a table with **13 rows**:

- **12 monthly rows** — one for each month of the year, so you can enter the
  fee for members joining in that month
- **1 renewals row** — the fee charged when existing members renew

An **auto-propagate toggle** is available: when turned on, entering a fee in one
month automatically fills in decreasing amounts for subsequent months, saving
you from typing each one individually.

> **Tip:** Make sure your fees are set up before the renewal season begins.
> Changes to fees take effect for new transactions but do not alter fees
> already charged.
