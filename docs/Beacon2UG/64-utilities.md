# 64. Utilities

The **Utilities** page provides access to a collection of administrative tools
for maintaining and checking your Beacon2 data.

To open it, click **Utilities** in the **Misc** section of the Home page.

![The Utilities page](images/utilities.png)

---

## Available tools

The page displays a menu of utility tools [A]. Each tool is shown with a
brief description of what it does. Currently the following tool is available:

- **Validate member data** [B] -- scans your member records for common data
  problems such as missing fields, invalid email addresses, or inconsistent
  dates. See [Section 15](15-validate-member-data.md) for full details.

> **Note:** This page will expand over time as additional utilities are added
> to Beacon2.

---

## Privilege requirements

Each tool on this page requires its own privilege. If you do not have the
required privilege for a particular tool, it will appear **greyed out** and
you will not be able to click it. Contact your Site Administrator if you need
access -- see [Section 50](50-roles-privileges.md) for more about roles and
privileges.
