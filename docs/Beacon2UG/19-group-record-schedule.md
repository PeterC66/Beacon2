# 19. Group Record -- Schedule

The **Schedule** tab of the Group Record lets you manage the group's
upcoming events -- meetings, outings, talks, and anything else that appears
on the calendar. You can add individual events or create a series of
recurring events in one go.

> **Note:** If **SiteWorks** integration is activated in System Settings, the
> Schedule tab is hidden because scheduling is managed through SiteWorks
> instead.

![The Group Record -- Schedule tab](images/19-group-record-schedule.png)

---

## The events table

The table lists all scheduled events for the group, with the following
columns:

| Column | Description |
|--------|-------------|
| **Select** | A checkbox for selecting events for bulk actions |
| **Date** [A] | The date of the event |
| **Time** [B] | The start time |
| **Venue** [C] | Where the event takes place |
| **Topic** [D] | A short description of the event (e.g. "Guest speaker: Jane Smith") |

Events are sorted by date, with the nearest event at the top.

---

## Show Detail toggle

Tick the **Show Detail** checkbox [E] above the table to expand each event
row and display additional information -- including end time, contact
details, and the full event description. Untick it to return to the compact
view.

---

## Adding a new event

Click the **Add Event** button [F] to open the event form. Fill in the
following fields:

| Field | Description |
|-------|-------------|
| **Date** [G] | The date of the event (required) |
| **Start time** [H] | When the event begins |
| **End time** [I] | When the event finishes |
| **Venue** [J] | Select a venue from the dropdown (venues are managed in [Section 21: Venues](21-venues.md)) |
| **Topic** [K] | A short title or description for the event |
| **Contact** [L] | The name or phone number of someone members can contact about this event |
| **Details** [M] | A longer description with any additional information |

### Recurrence options

If the group meets on a regular basis, you do not need to add each event
individually. Use the recurrence controls to create a series:

- **Every** [N] -- enter a number (e.g. 1, 2, 4)
- **Period** [O] -- choose **days**, **weeks**, or **months**
- **Until** [P] -- pick the end date for the series

For example, setting "Every **2** **weeks** until **31 December 2026**"
creates a fortnightly event from the start date through to the end of the
year.

Click **Save** to add the event (or series of events) to the schedule.

---

## Editing an existing event

Click on an event row in the table to open it for inline editing. Change any
of the fields and click **Save** to apply your changes, or **Cancel** to
discard them.

---

## Deleting events

### Single event

Open an event for editing and click **Delete** to remove it.

### Bulk delete

To remove several events at once:

1. Tick the checkboxes [Q] next to the events you want to delete.
2. Click the **Delete selected** button [R] below the table.
3. Confirm when prompted.

This is particularly useful for clearing out old events or removing a series
that is no longer needed.

---

## Downloading the schedule

Click **Download PDF** [S] to generate a printable PDF of the group's
schedule. This is handy for posting on a noticeboard or handing out at a
meeting.
