# 31. Groups Statement

The Groups Statement provides a summary of financial activity for your u3a's
interest groups. It is independent from the main finance ledger — it draws on
group-level ledger entries rather than the main account transactions.

To open it, click **Groups statement** on the **Home** page.

![Groups Statement screen](images/31-groups-statement.png)

---

## Filtering the report

### Date range [A]

Use the **From** and **To** date fields to choose the period you want to report
on. The statement will include all group ledger activity within those dates.

### Show individual transactions [B]

Tick the **Show individual transactions** checkbox if you want to see the detail
behind the summary figures. When ticked, the table expands to show each
transaction nested under its group. When unticked, you see just the totals per
group.

---

## The groups table [C]

The table shows one row per group with the following columns:

| Column | Description |
|--------|-------------|
| **Group** | The name of the interest group |
| **Balance b/f** | Balance brought forward — the group's balance at the start of the selected period |
| **Income** | Total income for the group during the period |
| **Expenditure** | Total expenditure for the group during the period |
| **Balance** | The group's balance at the end of the period |
| **Status** | The group's current status (e.g. Active, Suspended) |

### Nested transaction details

When **Show individual transactions** is ticked, each group row expands to show
the individual transactions that make up the totals. This lets you see exactly
what happened — for example, £30 in room hire and £15 in materials for the Art
group.

### Totals row [D]

At the bottom of the table, a **Totals** row sums up the balance brought
forward, income, expenditure, and closing balance across all groups.

---

## Downloading the statement [E]

Click **Download as Excel** to export the groups statement as a spreadsheet.
This is useful for sharing with group leaders or presenting at committee
meetings.
