# 4. The Home Page

After logging in (and changing your password if required), you arrive at the
**Home page**. This is your main starting point for everything in Beacon2.

![The Beacon2 Home page](images/home-page.png)

## The Administration heading

At the top of the page you will see the heading **Administration** [A], along
with your name and a **Log Out** link [B]. Click **Log Out** when you have
finished your session.

## The main menu grid

The centre of the page is a grid of links organised into five columns (on a
desktop) or five collapsible sections (on a phone or tablet):

| Column | What you will find there |
|--------|--------------------------|
| **Membership** [C] | Members list, Add new member, Membership renewals, Recent members, Non-renewals, Membership cards, Addresses export, Statistics |
| **Groups** [D] | Groups list, Venues, Faculties (subject categories), Calendar |
| **Finance** [E] | Ledger views (by account, category, or group), Add transaction, Transfer money, Credit batches, Reconcile account, Financial statement, Groups statement, Gift Aid declaration |
| **Misc** [F] | Audit log, Gift Aid log, u3a Officers, Public links, Data export & backup, E-mail delivery, E-mail unblocker, Personal preferences, Utilities |
| **Set up** [G] | System users, Roles and privileges, System settings, System messages, Finance accounts, Finance categories, Membership classes, Member statuses, Poll, Custom fields |

> **Tip:** Hover your mouse over any link in the grid to see a short
> description of what that page does.

### Greyed-out items

If a link appears in grey rather than blue, it means you do not have the
required privilege to access that page. Your Site Administrator can adjust
your privileges if needed (see [Section 50](50-roles-privileges.md)).

## External links

Below the main grid there is a row of links to external u3a Beacon resources
[H]:

- **u3a Beacon Users' Forum** -- community support forum for Beacon users
- **Beacon User Guide** -- the online reference guide
- **Beacon Website** -- news, training, and support from the Beacon team

## Public website links

A second row labelled **Public website links** [I] provides quick access to
your u3a's public-facing pages:

- **Join [your u3a] now!** -- the online joining page for prospective members
- **Members Portal** -- the self-service portal for existing members
- **Public groups list** -- a publicly visible list of your interest groups
- **Public calendar** -- a publicly visible calendar of group meetings

Some of these links may be greyed out if the corresponding feature has not yet
been configured in your u3a's settings.

## Messages

Below the links panel, Beacon2 may display one or two messages:

- **System-wide message** -- set by the Beacon support team and shown to all
  u3a organisations (for example, planned maintenance notices).
- **Home page notice** -- set by your own Site Administrator (for example,
  reminders about renewals or upcoming events). This is configured in
  [System Settings](48-system-settings.md).

If no messages have been set, this area will not appear.
