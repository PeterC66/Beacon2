# 13. Addresses Export

The **Addresses export** screen is a specialised tool for exporting member addresses.
Its key feature is that it automatically combines partners living at the same address
into a single row — so "Mr J Smith & Mrs A Smith" appear on one label rather than two.

To open it, click **Addresses export** on the Home page.

![Addresses export screen](images/13-addresses-export.png)

---

## Format options

Use the format selector [A] at the top of the screen to choose your output format:

| Format | Description |
|--------|-------------|
| **TAM** | An Excel file formatted for Third Age Matters magazine distribution. Use this when submitting your membership list for TAM delivery. |
| **Labels** | A PDF of address labels ready for printing. Configure the label layout in the settings panel (see below). |
| **Excel** | A standard Excel spreadsheet of addresses. |
| **CSV** | Comma-separated values file — useful for importing into other software. |
| **TSV** | Tab-separated values file — similar to CSV but using tabs as delimiters. |

---

## Filters

Use the filter controls [B] to narrow down which members are included in the export:

- **Status** — Choose one or more membership statuses (e.g. Current, Life).
- **Class** — Filter by membership class (e.g. Individual, Joint).
- **Poll** — Filter by a specific poll.
- **Group** — Filter by group membership (e.g. only members of a particular group).

Filters can be combined. For example, you could export labels for all Current
Individual members who belong to the "Walking" group.

---

## Label settings panel

When you select the **Labels** format, a settings panel [C] appears where you can
configure the label layout to match your label sheets:

| Setting | What it controls |
|---------|-----------------|
| **Columns** | Number of labels across the page (e.g. 2 or 3) |
| **Rows** | Number of labels down the page |
| **Label width** | Width of each label in mm |
| **Label height** | Height of each label in mm |
| **Left offset** | Left margin before the first column, in mm |
| **Top offset** | Top margin before the first row, in mm |
| **Column gap** | Horizontal gap between label columns, in mm |
| **Row gap** | Vertical gap between label rows, in mm |
| **Font size** | Size of the address text in points |

> **Tip:** Click **Save settings** [D] to store your label dimensions in your
> browser so you do not have to re-enter them each time. These settings are saved
> locally — they are personal to you and your browser, not shared with other users.

Common label sheets (such as Avery L7160 or L7163) have well-known dimensions.
Check the packaging of your label sheets for the correct measurements.

---

## Selecting members

The member table [E] shows all members matching your filters. Each row shows the
combined name (for partners at the same address) and address details.

- Use the **checkboxes** on the left to select specific members.
- Use the **Select all** checkbox in the header to select everyone.
- The selection count is shown above the table.

---

## Downloading and printing

Once you have chosen your format, applied filters, and selected members:

- Click **Download** [F] to save the file (Excel, CSV, TSV, or PDF labels) to
  your computer.
- For the **Labels** format, you can also click **Print** [G] to send the labels
  directly to your printer.

> **Tip:** Always do a test print on plain paper first to check alignment before
> printing on actual label sheets.

---

## Typical workflow — printing address labels

1. Click **Addresses export** on the Home page.
2. Select **Labels** as the format.
3. Set your filters (e.g. Status = Current).
4. Configure the label settings to match your label sheets (or load saved settings).
5. Select all members (or just those you need).
6. Click **Print** or **Download** to produce the labels.
7. Do a test print on plain paper, then print on labels.
