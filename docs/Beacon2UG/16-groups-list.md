# 16. The Groups List

The Groups List is your starting point for managing the interest groups in your
u3a. It shows every group at a glance, with handy filters so you can quickly
find the one you need.

## Opening the Groups List

From the **Home** page, click **Groups** in the navigation bar. You will see
a table listing all of your u3a's interest groups.

![The Groups List page](images/16-groups-list.png)

## What the list shows

Each row in the table shows the following information:

| Column | Description |
|--------|-------------|
| **Group** [A] | The name of the group (click to open the group record) |
| **When** [B] | A short description of when the group meets (e.g. "2nd Thursday at 2pm") |
| **Leader(s)** [C] | The name(s) of the group leader(s) |
| **Members** [D] | The number of members currently in the group |
| **Status** [E] | Whether the group is active or inactive (shown only when the active-only filter is unticked) |
| **Faculty** [F] | The faculty (subject category) the group belongs to, if any |

The columns with clickable headers can be **sorted** -- just click a column
heading to sort by that column. Click again to reverse the sort order.

## Filtering the list

Above the table you will find filtering controls that help you narrow down the
list:

### Active-only checkbox

By default the **Show active only** checkbox [G] is ticked, which means only
active groups are displayed. Untick it to include inactive groups as well --
inactive groups are labelled with a red "(inactive)" marker next to their name.

### Faculty dropdown

If your u3a uses faculties (subject categories), a **Faculty** dropdown [H]
appears. Select a faculty to show only groups belonging to that category, or
choose "All faculties" to see everything.

### Alphabet navigation

Below the filters is a row of letter buttons **A** through **Z**, plus an
**All** button [I]. Click a letter to show only groups whose name starts with
that letter. Click the same letter again (or click **All**) to clear the
filter. This is especially useful if your u3a has a large number of groups.

## Opening a group

Click the **group name** link in any row to open that group's record, where
you can view and edit its details, members, schedule, and ledger.

Alternatively, click the small **Edit** link on the right-hand side of the row.

## Adding a new group

If you have the appropriate permission, an **Add New Group** link appears in
the navigation bar at the top (and bottom) of the page. Click it to open a
blank group record ready for you to fill in.

> **Note:** The Add New Group link is only visible to users who have been
> granted the privilege to create group records.

## Count display

Just above the table, a small line shows how many groups match your current
filters -- for example, "24 groups".
