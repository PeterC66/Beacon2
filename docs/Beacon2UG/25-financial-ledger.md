# 25. The Financial Ledger

The Financial Ledger is where you see all the money coming in and going out of
your u3a's accounts. Beacon2 gives you **three different views** of exactly the
same transaction data, so you can look at your finances from whichever angle is
most useful.

To open the ledger, click one of the following on the **Home** page:

- **Ledger (by account)**
- **Ledger (by category)**
- **Ledger (by group)**

![The Financial Ledger — account view](images/25-financial-ledger.png)

---

## The three views

All three views show the same underlying transactions — they just organise them
differently.

### Ledger (by account) [A]

Select an **Account** from the dropdown to see every transaction in that account.
This view shows a **running balance** and each transaction's **cleared status**,
making it ideal for day-to-day bookkeeping and reconciliation.

### Ledger (by category) [B]

Select a **Category** from the dropdown to see all transactions allocated to that
category, regardless of which account they sit in. Use this when you want to
answer questions like "How much have we spent on room hire this year?"

### Ledger (by group) [C]

Select a **Group** from the dropdown to see transactions tagged against that group.
You can optionally **filter by group** to narrow the display further. This is
useful for group leaders and treasurers who want to review their group's finances.

---

## Choosing a year

Use the **Year** selector [D] at the top of the page to choose which financial
year to display. Beacon2 keeps up to **six years** of history, so you can look
back at previous years without needing a separate archive.

---

## Table columns

The ledger table includes the following columns (not all appear in every view):

| Column | Description |
|--------|-------------|
| **No** | Transaction number — click to open the full transaction record |
| **Date** | The date of the transaction |
| **Detail** | A short description of what the transaction was for |
| **From / To** | Who the money came from or was paid to |
| **Method** | The payment method (e.g. BACS, cheque, cash) |
| **In** | Amount received (for income transactions) |
| **Out** | Amount paid out (for payment transactions) |
| **Balance** | Running balance — shown in the **account view** only |
| **Refund** | If the transaction has been refunded, a link to the refund transaction |
| **Batch** | If the transaction belongs to a credit batch, the batch reference |
| **Cleared** | Whether the transaction has been reconciled (tick or blank) |

> **Tip:** Click a **transaction number** to open the full Transaction Record
> (see [Section 26](26-transaction-record.md)). Click a **batch reference** to
> jump to that batch in Credit Batches (see [Section 28](28-credit-batches.md)).
> Click a **refund reference** to navigate to the linked refund transaction.

---

## Bulk actions (account view)

In the **Ledger (by account)** view, you can select one or more transactions
using the checkboxes on the left, then use the bulk action buttons to:

- **Mark as Pending** — move selected transactions to pending status (they will
  be excluded from the running balance and financial statement)
- **Confirm** — confirm selected pending transactions so they are included in
  the running balance

These actions are only available when the account has pending transactions
enabled (see [Section 34](34-pending-transactions.md)).

---

## Navigating the ledger

- Click any **transaction number** to view or edit that transaction.
- Click a **batch reference** to open the credit batch.
- Click a **refund reference** to jump to the linked refund or original
  transaction.
- Use the **Year** selector to move between financial years.
- Switch between the three views using the links at the top of the page.
