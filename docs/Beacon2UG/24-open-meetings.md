# 24. Open Meetings

Open Meetings are events that are not tied to any specific interest group.
They cover u3a-wide activities such as AGMs, social events, guest lectures,
and other gatherings that any member can attend.

Open Meeting events appear alongside group events in the
[Calendar](23-calendar.md), but they are managed on their own dedicated
page.

## Accessing Open Meetings

From the **Calendar** page, click the **Open Meetings** link in the
navigation. The Open Meetings page opens, showing all open meeting events.

![The Open Meetings page](images/24-open-meetings.png)

---

## The events table

The table lists all open meeting events in date order, with the following
columns:

| Column | Description |
|--------|-------------|
| **Select** | A checkbox for selecting events for bulk deletion |
| **Date** [A] | The date of the event |
| **Time** [B] | The start time |
| **Venue** [C] | Where the event takes place |
| **Topic** [D] | A short description of the event |

---

## Show Detail toggle

Tick the **Show Detail** checkbox [E] above the table to expand each event
row and show additional information -- including end time, enquiries contact,
and the full event description. Untick it to return to the compact view.

---

## Adding an open meeting event

Click the **Add Event** button [F] to open the event form. Fill in the
following fields:

| Field | Description |
|-------|-------------|
| **Date** [G] | The date of the event (required) |
| **Start time** [H] | When the event begins |
| **End time** [I] | When the event finishes |
| **Venue** [J] | Select a venue from the dropdown (venues are managed in [Section 21: Venues](21-venues.md)) |
| **Topic** [K] | A short title or description for the event |
| **Enquiries** [L] | The name or phone number of someone members can contact about this event |
| **Details** [M] | A longer description with any additional information |
| **Private** [N] | Tick this checkbox to exclude the event from the public calendar -- private events are only visible to logged-in Beacon2 users |

### Recurrence options

If you hold regular open meetings (for example a monthly social), use the
recurrence controls to create a series:

- **Every** [O] -- enter a number (e.g. 1, 2, 3)
- **Period** [P] -- choose **days**, **weeks**, or **months**
- **Until** [Q] -- pick the end date for the series

For example, setting "Every **1** **month** until **31 December 2026**"
creates a monthly event from the start date through to the end of the year.

Click **Save** to add the event (or series of events).

---

## Editing an existing event

Click on an event row in the table to open it for inline editing. Change any
of the fields and click **Save** to apply your changes, or **Cancel** to
discard them.

---

## Deleting events

### Single event

Open an event for editing and click **Delete** to remove it.

### Bulk delete

To remove several events at once:

1. Tick the checkboxes [R] next to the events you want to delete.
2. Click the **Delete selected** button [S] below the table.
3. Confirm when prompted.

---

## How Open Meetings appear in the Calendar

Open meeting events are shown in the [Calendar](23-calendar.md) alongside
group events. They appear with no group name, making them easy to identify.
Events marked as **Private** are excluded from any public-facing calendar
(such as the Members Portal or website) but still appear in the
administrative Calendar within Beacon2.
